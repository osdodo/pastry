# Workflow 使用说明

本文档说明如何在 Pastry 中创建和运行工作流。

## 什么是工作流

工作流是一个节点图：由触发器启动，并按连接顺序处理数据。

## 节点类型

- **热键**：通过全局快捷键触发工作流。
- **脚本**：使用 JavaScript 转换数据。
- **剪贴板**：读取或写入系统剪贴板。
- **文件写入**：将输出内容写入本地文件。

## 基本配置流程

1. 打开工作流页面。
2. 新建一个工作流。
3. 添加节点并建立连接。
4. 配置每个节点的输入与输出。
5. 启用该工作流。

## 触发与执行

- 如果包含 `热键` 节点，可通过配置的快捷键触发。
- 也可在工作流列表或编辑器中手动执行（若界面支持）。

## 实用示例

`剪贴板读取 -> 脚本 -> 剪贴板写入`

- 从剪贴板读取文本。
- 在脚本节点中转换（如 Base64、哈希、格式化）。
- 将结果写回剪贴板。
