# 局域网同步使用指南

本文档说明如何使用 Pastry 的局域网同步功能。

## 功能概览

局域网同步允许同一局域网内的设备通过浏览器访问并同步剪贴板内容。

- 默认访问地址格式：`http://<你的局域网IP>:8080`
- 默认端口：`8080`

## 开启局域网同步

1. 打开 Pastry 设置。
2. 开启 **局域网同步**。
3. 确认界面显示二维码和访问地址。

如果显示“未检测到可用局域网地址”，请检查 Wi-Fi / 网络连接后重试。

## 其他设备如何连接

1. 确保手机/平板/电脑与 Pastry 所在设备处于同一局域网。
2. 扫描 Pastry 中显示的二维码，或手动打开访问地址。
3. 在网页端进行剪贴板内容的发送与接收。

## 网页端能力

- **接收内容**：查看最新同步内容，并支持一键复制。
- **发送内容**：向 Pastry 推送文本、链接、代码片段和图片。
- **实时同步**：通过 WebSocket 实时更新状态与内容。

## 健康检查

可访问 `http://<你的局域网IP>:8080/health` 进行连通性检查。

- 服务正常时会返回 `ok`。

## 常见问题排查

- 无法打开页面：先确认同一局域网，再检查防火墙/路由器是否拦截 `8080` 端口。
- 扫码后打不开：尝试复制地址到浏览器手动访问。
- 内容不同步：刷新网页，并确认 Pastry 的局域网同步仍处于开启状态。

## 安全说明

- 当前局域网同步服务监听在 `0.0.0.0:8080`。
- 局域网中可访问该地址的设备都可能访问同步页面。
- 建议仅在可信网络中使用，不需要时及时关闭该功能。
