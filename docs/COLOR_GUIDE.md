# Color Picker Guide

This document explains the color tools in Pastry.

## Supported Inputs

Pastry can detect common color formats from clipboard content, including:

- HEX (for example `#FF5733`)
- RGB / RGBA
- Other common textual color formats supported by the app parser

## What You Can Do

- Preview detected colors.
- Pick and adjust colors visually.
- Copy normalized color values for reuse.

## Typical Workflow

1. Copy a color string (or content containing one).
2. Open Pastry and select the color tool.
3. Adjust the color if needed.
4. Copy the desired output format.
